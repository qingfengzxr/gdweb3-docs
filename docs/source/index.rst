欢迎来到 GDWeb3 的文档！
==========================

**GDWeb3** 是一个为 Godot 定制的模块，允许用户使用 GDScript 脚本语言来与区块链网络进行交互，从而方便Godot生态中的游戏开发者们开发GameFi游戏或者全链游戏。

.. image:: _static/image.png
   :alt: alternate text
   :align: center

描述
----
**GDWeb3** 旨在帮助 Godot 游戏开发者轻松与区块链网络交互，赋能他们开发 Gamefi 和 FOCG（全链游戏）。

然而，同时支持所有区块链网络是不现实的。在这个阶段，我们主要关注 `Optimism Network <https://www.optimism.io/>`_。`Optimism` 作为 ETH Layer 2 解决方案的性能非常出色，并且拥有强大的生态系统。

* 我们相信，支持一个有前景的网络将吸引游戏开发者继续构建。

* 我们相信，游戏开发者独特的创造力和无限的想法将加速 web3 游戏行业的发展。

能够为他们提供帮助，并且也许有一天能玩到有趣的 FOCG 游戏，是我们维护这个项目的乐趣所在。


关于 Godot 引擎
---------------
Godot 引擎是一个功能丰富的跨平台游戏引擎，允许用户从统一的界面创建 2D 和 3D 游戏。它提供了一套全面的常用工具，让用户可以专注于制作游戏，而无需重新发明轮子。游戏可以一键导出到各种平台，包括主要的桌面平台（Linux、macOS、Windows）、移动平台（Android、iOS）、基于网络的平台和控制台。

Godot 完全免费且开源，采用宽松的 MIT 许可证。用户的游戏完全属于他们自己，包括引擎代码的最后一行。Godot 的开发完全独立且由社区驱动，使用户能够根据自己的期望塑造引擎。它由 Godot 基金会支持，这是一个非营利组织。

`点击这里了解更多关于 Godot 引擎的详细信息。 <https://godotengine.org>`_


支持的 Godot 版本
------------------
在这个阶段，我们基于 **4.3.0-stable** 版本进行开发。目前，我们只保证向上兼容。因此，当您使用从该项目编译的程序在旧的游戏项目上进行扩展开发时，请注意您最初使用的 Godot 引擎程序的版本号。


.. note::

   该项目正在积极开发中。
